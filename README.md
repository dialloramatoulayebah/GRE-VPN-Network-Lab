# 🌐 GRE Tunnel VPN Lab – Toronto ↔ Montréal

## 📌 Project Overview
This project demonstrates the implementation of a **GRE (Generic Routing Encapsulation) tunnel** between two remote sites (Toronto and Montréal) using Cisco Packet Tracer.

The goal is to simulate a real-world WAN environment where two private LAN networks communicate securely over an ISP network.

---

## 🧠 Skills Demonstrated
- Network configuration (Cisco routers & switches)
- WAN and ISP simulation
- GRE Tunnel configuration
- Dynamic routing using EIGRP
- VLAN segmentation (Router-on-a-stick)
- Troubleshooting and network testing

---

## 🔑 Key Skills for IT Support

---

## 🏗️ Network Architecture
- Two sites: Toronto (R1) and Montréal (R2)
- ISP network acting as intermediary
- GRE Tunnel over public IP addresses
- VLAN 10 & VLAN 20 for network segmentation

---

## 🔧 Technologies Used
- Cisco Packet Tracer
- GRE (Generic Routing Encapsulation)
- EIGRP Routing Protocol
- VLAN & Trunking

---

## ⚙️ Configuration Steps
1. Configure ISP network connectivity
2. Configure base router settings (R1 & R2)
3. Create GRE Tunnel interfaces
4. Configure EIGRP for dynamic routing
5. Configure VLANs (Router-on-a-stick)
6. Test connectivity (Ping)
